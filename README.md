SplitMetrics App v1.0
============

## Installation

Before running any Gulp tasks:

1. Check out this repository
2. Ensure you have node installed
3. Run `npm install` in the root directory (this will install bower dependencies too)
4. Run `bower install`.
5. Run `gulp watch`.
